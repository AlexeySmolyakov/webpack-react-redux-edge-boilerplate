# React boilerplate

Start developing fast and from the cutting edge.  
Fill free to contribute.

## Main features

All packages have latest versions.  
This boilerplate is 100% working and maintained and is used for creating all new projects.    

## Prerequisites

- NodeJS
- npm

## Installing

```
npm install
npm start
```
